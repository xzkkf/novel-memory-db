 剧情记忆数据库

> 一部218章网络小说的结构化处理引擎，为漫剧改编提供精确、可检索的剧情基底。

---

## 项目简介

本项目将《月光兔传奇》全本218章小说进行自动化结构化处理，拆解为 **5,212 个独立场景**，并为每个场景生成剧情记忆卡。所有数据以 JSON 格式存储，支持按章节、角色、事件类型等多维度查询，可直接服务于漫剧/动画改编的前期数据准备工作。

## 文件说明

| 文件 | 说明 |
|------|------|
| `novel_memory_db.json` | 核心数据库（7.8MB），含全部 5,212 张记忆卡 |
| `query_interface.py` | Python 查询接口：前情提要、角色线、关键词搜索 |
| `process_novel.py` | 小说处理脚本，可复用于其他小说 |
| `trailer_scenes/` | 漫剧预告片素材：15个场景 + 即梦AI提示词 + 30秒分镜 |
| `剧情记忆数据库报告.md` | 完整章节-场景树（1.4MB） |
| `关键剧情节点时间线.md` | 870个核心事件时间线 |

## 快速使用

```python
from query_interface import get_story_recap, get_character_arc, search_by_keyword

# 获取第100章之前的关键剧情
recap = get_story_recap(100)

# 追踪月光兔的剧情线
arc = get_character_arc("月光兔")

# 搜索关键词
results = search_by_keyword("抱月玉槌")
```

## 漫剧预告片

`trailer_scenes/` 目录包含为 30 秒漫剧预告片提取的素材：

- **15 个高视觉冲击力场景** — 涵盖穿越觉醒、战斗高潮、传说登场、阵容集结
- **即梦AI提示词** — 每条 200-280 字符，日系动漫画风
- **30秒分镜时间线** — 含画面描述、配音关键词、音效提示
