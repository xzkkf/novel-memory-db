## 使用说明

### 查询前情提要
```python
from query_interface import get_story_recap
recap = get_story_recap(100)  # 获取第100章之前的关键剧情
```

### 查询角色剧情线
```python
from query_interface import get_character_arc
arc = get_character_arc("月光兔")  # 获取月光兔的关键剧情
```

### 关键词搜索
```python
from query_interface import search_by_keyword
results = search_by_keyword("抱月玉槌")  # 搜索相关场景
```

### 获取章节场景
```python
from query_interface import get_chapter_scenes
scenes = get_chapter_scenes(42)  # 获取第42章的所有场景
```
